Полнофункциональный плагин Bedwars для Minecraft 1.8.8, вдохновлённый системой Hypixel.

## Описание

Bedwars - это PvP мини-игра, где команды защищают свою кровать и пытаются уничтожить кровати противников. Игроки собирают ресурсы с генераторов, покупают предметы в магазине, улучшают свою команду и сражаются за победу.

## Особенности

### Игровые режимы
- **Solo** (1v1v1v1v1v1v1v1) - 8 команд по 1 игроку
- **Doubles** (2v2v2v2v2v2v2v2) - 8 команд по 2 игрока
- **3v3v3v3** (Trio) - 8 команд по 3 игрока
- **4v4v4v4** (Squad) - 8 команд по 4 игрока

### Система арен
- Автоматический выбор арены по загруженности
- Поддержка нескольких арен одного типа
- Сохранение/загрузка конфигурации арен в YAML
- Настройка спавнов команд, кроватей и генераторов

### Генераторы ресурсов
- **Железо** - генерируется каждую секунду
- **Золото** - генерируется каждые 8 секунд
- **Алмазы** - генерируется каждые 30 секунд
- **Изумруды** - генерируется каждые 60 секунд
- Автоматическое улучшение генераторов по времени игры

### Магазин
Полноценный GUI магазин с категориями:
- **Блоки** - шерсть, глина, стекло, эндер-камень, обсидиан
- **Оружие** - мечи, палка отбрасывания
- **Броня** - кольчужная, железная, алмазная
- **Инструменты** - кирки, топоры, ножницы с автоапгрейдом
- **Луки** - обычный лук, зачарованные луки, стрелы
- **Зелья** - скорость, прыгучесть, невидимость
- **Особое** - золотое яблоко, TNT, эндер-жемчуг, огненный шар

### Улучшения команды
- **Заточенные мечи** - Заточенность I на все мечи команды
- **Усиленная броня** - Защита I-IV на броню команды
- **Спешка** - Спешка II на базе команды
- **Ловушки** - Эффекты при вторжении врагов

### События игры
- **2:00** - Алмазы II (генераторы работают быстрее)
- **6:00** - Изумруды II (генераторы работают быстрее)
- **10:00** - Алмазы III (генераторы работают ещё быстрее)
- **15:00** - Внезапная смерть (все кровати уничтожаются)

### Дополнительные функции
- Компас-трекер для поиска врагов
- Система статистики игроков (убийства, смерти, победы)
- Автоматическое распределение по командам
- Защита от урона по союзникам
- Система наблюдателей
- Автоматический респавн с выдачей предметов

## Команды

### Основные команды
- `/bw join <арена>` - присоединиться к арене
- `/bw leave` - покинуть арену
- `/bw list` - список доступных арен

### Быстрый вход
- `/bw solo` - автоматический вход в Solo режим
- `/bw doubles` - автоматический вход в Doubles
- `/bw trio` - автоматический вход в 3v3v3v3
- `/bw squad` - автоматический вход в 4v4v4v4

### Настройка арен (требуется OP)
- `/setup <арена> create <тип>` - создать арену
- `/setup <арена> lobby` - установить спавн лобби
- `/setup <арена> spectator` - установить спавн наблюдателей
- `/setup <арена> teamspawn <цвет>` - установить спавн команды
- `/setup <арена> bed <цвет>` - установить кровать команды
- `/setup <арена> generator <тип>` - добавить генератор (IRON/GOLD/DIAMOND/EMERALD)
- `/setup <арена> save` - сохранить арену
- `/setup <арена> delete` - удалить арену

### Цвета команд
`RED`, `BLUE`, `GREEN`, `YELLOW`, `AQUA`, `WHITE`, `PINK`, `GRAY`

## Установка

1. Скачайте `Bedwars-1.0.jar` из [Releases](../../releases)
2. Поместите файл в папку `plugins` вашего сервера Spigot/Bukkit 1.8.8
3. Перезапустите сервер
4. Настройте арены с помощью команд `/setup`

## Требования

- **Сервер:** Spigot/Bukkit 1.8.8
- **Java:** 8 или выше
- **Зависимости:** Нет (полностью автономный плагин)
